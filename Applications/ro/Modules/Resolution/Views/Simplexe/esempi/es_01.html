Analisi

Il problema introdotto è

max z = 2 x1 + x2

Soggetto a

1) x1 - 2 x2 =< 2
2) -3 x1 + x2 =< 3
3) x1 + x2 =< 4
        xi >= 0     per i = 1,...,2

Il problema espresso in FORMA STANDARD

min z = -2 x1 - x2

Soggetto a

1) x1 - 2 x2 + x3 = 2
2) -3 x1 + x2 + x4 = 3
3) x1 + x2 + x5 = 4
        xi >= 0     per i = 1,...,5

La FORMA CANONICA coincide con quella STANDARD.
