Analisi

Il problema introdotto è

min z = - 3/4 x4 - 1/2 x6 + 6 x7

Soggetto a

1) x1 + 1/4 x4 - 8 x5 - x6 + 9 x7 = 0
2) + x2 + 1/2 x4 - 12 x5 - 1/2 x6 + 3 x7 =< 0
3) + x3 + x6 >= 1
        xi >= 0     per i = 1,...,7

Il problema espresso in FORMA STANDARD

min z = - 3/4 x4 - 1/2 x6 + 6 x7

Soggetto a

1) x1 + 1/4 x4 - 8 x5 - x6 + 9 x7 = 0
2) + x2 + 1/2 x4 - 12 x5 - 1/2 x6 + 3 x7 + x8 = 0
3) + x3 + x6 - x9 = 1
        xi >= 0     per i = 1,...,9

La FORMA CANONICA coincide con quella STANDARD.
