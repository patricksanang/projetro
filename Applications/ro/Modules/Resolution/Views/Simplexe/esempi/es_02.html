Analisi

Il problema introdotto è

min z = - x2

Soggetto a

1) 2 x1 + x2 =< 2
2) - x1 + x2 =< 0
        xi >= 0     per i = 1,...,2

Il problema espresso in FORMA STANDARD

min z = - x2

Soggetto a

1) 2 x1 + x2 + x3 = 2
2) - x1 + x2 + x4 = 0
        xi >= 0     per i = 1,...,4

La FORMA CANONICA coincide con quella STANDARD.

Metodo del SIMPLESSO

Tableau al passo 0:

x1 x2 x3 x4
r00 0 -1 0 0
r12 2 1 1 0
r20 -1 1 0 1
Indici di base: S = { 3, 4 }
Soluzione di base: z = 0
x1 = 0
x2 = 0
x3 = 2
x4 = 0
Soluzione migliorabile. L'algoritmo continua ad iterare.
Pivot in riga r2 colonna x2.

Tableau al passo 1:

x1 x2 x3 x4
r00 -1 0 0 1
r12 3 0 1 -1
r20 -1 1 0 1
Indici di base: S = { 3, 2 }
Soluzione di base: z = 0
x1 = 0
x2 = 0
x3 = 2
x4 = 0
Soluzione migliorabile. L'algoritmo continua ad iterare.
Pivot in riga r1 colonna x1.

Tableau al passo 2:

x1 x2 x3 x4
r02/3 0 0 1/3 2/3
r12/3 1 0 1/3 -1/3
r22/3 0 1 1/3 2/3
Indici di base: S = { 1, 2 }
Soluzione di base: z = -2/3
x1 = 2/3
x2 = 2/3
x3 = 0
x4 = 0
Soluzione ottima:     z* = -2/3,     x* = [ 2/3, 2/3, 0, 0 ]T
