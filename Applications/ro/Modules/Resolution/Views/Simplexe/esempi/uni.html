Analisi

Il problema introdotto è

max z = - x1 - x2 + x3 + x4

Soggetto a

1) x1 + x2 + x3 + x4 =< 1
2) x1 - x2 + x3 - x4 =< 1
3) x1 + x2 - x3 - x4 =< 1
4) x1 - x2 - x3 + x4 =< 1
5) - x1 - x2 - x3 - x4 =< 1
6) - x1 + x2 - x3 + x4 =< 1
7) - x1 - x2 + x3 + x4 =< 1
8) - x1 + x2 + x3 - x4 =< 1
        xi >= 0 e INTERI     per i = 1,...,4

Il problema espresso in FORMA STANDARD

min z = x1 + x2 - x3 - x4

Soggetto a

1) x1 + x2 + x3 + x4 + x5 = 1
2) x1 - x2 + x3 - x4 + x6 = 1
3) x1 + x2 - x3 - x4 + x7 = 1
4) x1 - x2 - x3 + x4 + x8 = 1
5) - x1 - x2 - x3 - x4 + x9 = 1
6) - x1 + x2 - x3 + x4 + x10 = 1
7) - x1 - x2 + x3 + x4 + x11 = 1
8) - x1 + x2 + x3 - x4 + x12 = 1
        xi >= 0 e INTERI     per i = 1,...,12

La FORMA CANONICA coincide con quella STANDARD.
